## Team School Spirit Squad - Personal Book Library Management Application

## Description
This is a Node.js backend application built with Express.js, GraphQL and Sequelize, designed to manage personal book libraries, reading circles and community interactions. The platform allows users to organize their readings, participate in book discussion groups and receive personalized notifications.

## Getting started  
1. Run `npm install`  

## Database setup  
Run the resfresh.sh script to create and seed the database.
